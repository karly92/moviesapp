Aplicación de Películas realizada con el Framework Ruby on Rails implementando el 
algoritmo de Jaccard y Clasificadores Bayesianos. 
En la parte de Jaccard encuentra las peliculas relacionadas respecto a su género.
En la parte de Clasificadores Bayesianos clasifica Preguntas tecnicas y comerciales.
